# 安全策略

## 报告漏洞

如果你认为在此仓库中发现了安全漏洞，可以通过[私有漏洞报告](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability)向我们报告。

如果你无法通过上述机制提交报告，或想要报告任何其他纪程服务或应用程序的漏洞，请发送电子邮件至 security@gitbutler.com。

**请勿通过公开的 GitHub issues、discussions、pull request 或任何其他公开可访问的形式报告安全漏洞。**

感谢你帮助维护纪程的安全。

## 奖励计划

纪程目前没有活跃的奖励计划。

## 受支持的版本

纪程目前不提供"长期支持"（LTS）版本。安全修复不会向后移植。为确保系统安全，请保持与我们的版本同步更新。
